# Support

## Getting help
For usage questions and non-security issues:
1. Review `README.md` and docs in `docs/`.
2. Open a GitHub issue using the most relevant template.

## Issue quality requirements
Provide:
- WordPress version
- PHP version
- plugin version/tag
- expected behavior
- actual behavior
- reproducible steps

## Security issues
Security reports must follow `SECURITY.md` and stay private until a fix is available.
